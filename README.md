# Расчёт параметров крепления горных выработок

Локальное веб-приложение на Flask для автоматического подбора параметров крепления горных выработок по категории нарушенности массива и размерам выработки.

## Возможности

- Загрузка входного файла: **CSV**, **Excel** (xlsx/xls), **JSON**
- Автоматическое сопоставление столбцов входных данных с нужными полями
- Предпросмотр данных прямо в браузере
- Выгрузка результата в **Excel (.xlsx)** с автошириной колонок и переносом текста

## Что рассчитывается

Для каждой выработки по категории нарушенности (Q) и ширине определяются:

| Поле | Описание |
|------|----------|
| Параметры временного крепления | Состав и описание временной крепи |
| Шаг анкерования | Сетка анкеров, например `1.1×1.1 м` |
| Глубина анкерования, м | Требуемая глубина заделки |
| Отставание временной крепи, м | Допустимое отставание от забоя |
| Параметры постоянного крепления | Состав и описание постоянной крепи |
| Отставание постоянного крепления, м | Допустимое отставание от забоя |

## Требования

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) — менеджер пакетов и окружений

## Установка и запуск (из исходников)

```bash
# 1. Установить uv (если ещё нет)
curl -LsSf https://astral.sh/uv/install.sh | sh

# 2. Создать окружение и установить зависимости
uv sync

# 3. Запустить приложение
uv run krep
```

Приложение автоматически откроет браузер по адресу `http://127.0.0.1:5000`.  
Чтобы остановить — закройте терминал или нажмите `Ctrl+C`.

## Альтернативный запуск (без установки команды)

```bash
uv run python -m krep_app
```

## Сборка в исполняемый файл

### macOS / Linux

```bash
./build.sh
# результат: dist/krep_app  (единый бинарник, ~50 МБ)
./dist/krep_app
```

### Windows

```bat
build_exe.bat
REM результат: dist\krep_app.exe
dist\krep_app.exe
```

> Для Windows-сборки нужно запускать команды именно на Windows — PyInstaller не поддерживает кросс-компиляцию между платформами.

## Входной файл

Файл должен содержать **минимум четыре столбца** (названия могут быть любыми — приложение автоматически их сопоставит, или можно указать вручную):

| Поле | Примеры названий столбцов |
|------|--------------------------|
| Название выработки | `name`, `Название`, `Название выработки` |
| Назначение выработки | `purpose`, `Назначение`, `Тип` |
| Категория нарушенности | `category`, `Категория`, `Q` |
| Ширина выработки, м | `width`, `Ширина`, `Ширина, м` |

Значения категории: `1а`, `1б`, `2а`, `2б`, `3а`, `3б`, `4`, `4а`, `4б`.  
Назначение: `Горно‑капитальные`, `Горно‑подготовительные`, `Нарезные`.
